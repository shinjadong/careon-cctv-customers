title: 개요

DroidRun 가이드에 오신 것을 환영합니다! 여기에서 다양한 LLM 프로바이더와 함께 DroidRun을 사용하고, 텔레메트리를 구성하는 등의 단계별 안내와 권장 사항을 확인할 수 있습니다. 각 가이드는 DroidRun을 처음 시작하든 고급 기능을 활용하려고 하든 최대한 활용할 수 있도록 설계되었습니다.

---

## [CLI 사용 가이드](./cli)
DroidRun 커맨드라인 인터페이스를 통해 자연어와 LLM 에이전트를 사용하여 Android 기기를 제어하는 방법을 배웁니다. 이 가이드의 내용:
- DroidRun 및 종속성 설치
- Android 기기 연결
- 다양한 LLM 프로바이더와 CLI 사용 (OpenAILike, Ollama, Gemini)
- 주요 CLI 옵션 및 플래그
- 일반적인 문제 해결

---

## [Gemini와 함께 DroidRun 사용하기](./gemini)
Google의 고급 클라우드 기반 LLM인 [Gemini](https://ai.google.dev/gemini-api/docs/)와 함께 DroidRun을 사용하는 방법을 배웁니다. 이 가이드의 내용:
- Gemini란 무엇이며 어떤 기능이 있는지
- Gemini API 액세스 설정
- 필요한 Python 패키지 설치
- 예시: Gemini로 Android 자동화하기
- 문제 해결 및 고급 구성 팁

---

## [OpenAILike와 함께 DroidRun 사용하기](./openailike)
OpenAILike 호환 LLM(예: OpenAI의 GPT 모델 및 호환 API)과 함께 DroidRun을 사용하는 방법을 배웁니다. 이 가이드의 내용:
- OpenAILike의 의미와 지원되는 프로바이더
- API 키 및 액세스 설정
- 필요한 Python 패키지 설치
- 예시: OpenAILike LLM으로 Android 자동화하기
- 문제 해결 및 고급 구성 팁

---

## [Ollama와 함께 DroidRun 사용하기](./ollama)
로컬에서 대규모 언어 모델을 실행하기 위한 오픈소스 플랫폼인 [Ollama](https://ollama.com/)와 DroidRun을 통합하는 방법을 알아봅니다. 이 가이드의 설명 내용:
- Ollama란 무엇이며 왜 사용하는지
- Ollama 설정 및 모델 다운로드
- 필요한 Python 패키지 설치
- 예시: 로컬 LLM으로 Android 자동화하기
- 최상의 성능을 위한 문제 해결 및 팁

---

## [텔레메트리 가이드](./telemetry.mdx)
DroidRun이 프레임워크 개선을 위해 익명화된 텔레메트리를 사용하는 방법을 이해합니다. 이 가이드의 설명 내용:
- 텔레메트리가 중요한 이유
- 어떤 데이터가 수집되고 (수집되지 않는지)
- 텔레메트리를 비활성화하거나 활성화하는 방법
- 개인정보 보호에 대한 우리의 약속
