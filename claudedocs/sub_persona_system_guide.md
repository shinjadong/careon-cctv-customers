# 세부 페르소나 시스템 가이드 (Sub-Persona System)

## 🎯 시스템 개요

**목적**: 위약금 해방(Un-carrier) 전략을 위한 **3-2 이탈/고통 고객** 정밀 타게팅

기존 5단계 페르소나(1~5)를 세부 분류(Sub-persona) 체계로 고도화하여, 특히 케어온의 핵심 전략인 **"위약금 대납"** 캠페인을 위한 타겟 고객을 정밀하게 식별합니다.

## 💡 Un-carrier 전략 핵심

### 전략 배경

**문제**:
- 블로그 방문자 월 2,000명
- 전환율 0.38% (처참)
- 원인: 위약금 공포 (50~60만 원)

**해결책**:
- T-Mobile의 "Un-carrier" 전략 벤치마킹
- "우리가 고객의 감옥(위약금)을 부순다"

### 제공 혜택

```
기존 장비 반납 + 케어온 36개월 신규 계약
= 위약금 최대 50만 원 지원 (월 렌탈료 차감)

예시:
- 기존 위약금: 48만 원
- 케어온 월 렌탈료: 15,000원
- 지원 방식: 매월 10,000원씩 차감 (48개월)
- 실제 부담: 월 5,000원
```

### 왜 가능한가?

1. **대기업 마진 없음**: 직영 시스템 (30~40% 마진 절감)
2. **장기 고객 확보**: 안정적 수익 구조
3. **중소기업 혁신**: 대기업이 못하는 파격 혜택

## 💾 데이터베이스 쿼리

### 3-2 키워드만 추출

```sql
SELECT k.keyword, kt.search_volume_total, kt.competition_level
FROM keywords_master k
LEFT JOIN keyword_timeseries kt ON k.keyword_id = kt.keyword_id
WHERE k.sub_persona_id = '3-2'
ORDER BY kt.search_volume_total DESC;
```

### 세부 페르소나별 통계

```sql
SELECT sp.sub_persona_id, sp.sub_persona_name,
       COUNT(k.keyword_id) as keyword_count,
       SUM(kt.search_volume_total) as total_volume
FROM sub_personas sp
LEFT JOIN keywords_master k ON sp.sub_persona_id = k.sub_persona_id
LEFT JOIN keyword_timeseries kt ON k.keyword_id = kt.keyword_id
GROUP BY sp.sub_persona_id
ORDER BY total_volume DESC;
```
