# 🚀 케어온 블로그 자동화 시스템 - 빠른 시작 가이드

## ✅ 시스템 완성!

**위약금 해방(Un-carrier) 전략**을 위한 블로그 자동화 시스템이 완성되었습니다!

## 🎯 핵심 기능

### 1. 세부 페르소나 시스템
- **1,692개 키워드**를 6개 세부 페르소나로 분류
- 🔥 **3-2 이탈/고통 고객 14개** 정밀 식별 (위약금 해방 타겟)

### 2. 인터랙티브 원고 생성
- 페르소나 선택 → 키워드 추천 → AI 프롬프트 자동 생성
- 3-2 전용 **위약금 해방 캠페인** 프롬프트

## 🚀 즉시 사용하기

### 블로그 원고 자동 생성


### 실행 흐름

```

3. 프롬프트 생성 완료!
   → Claude/ChatGPT에 복사해서 사용
```

## 💡 위약금 해방 전략

### 핵심 메시지
```
"사장님, 그 위약금 저희가 부담하겠습니다."

기존 장비 반납 + 케어온 36개월 신규 계약
= 위약금 최대 50만 원 지원 (월 렌탈료 차감)
```

### 예시
```
기존 위약금: 48만 원
케어온 월 렌탈료: 15,000원
지원 방식: 48개월 동안 매월 10,000원씩 차감
실제 부담: 월 5,000원
```

## 💬 FAQ

### Q1: 왜 3-2가 중요한가요?
**A**: 위약금 때문에 갇혀있는 고객은 **전환 의지가 높지만 해결책이 없어서 못 움직이는** 상태입니다. 케어온의 위약금 대납은 이들에게 **유일한 탈출구**가 됩니다.

### Q2: 다른 페르소나는 어떻게 하나요?
**A**: 같은 스크립트로 1-1, 2-1, 3-1, 4-1, 5-1도 원고 생성 가능합니다. 각 페르소나별 맞춤 프롬프트가 자동 생성됩니다.

### Q3: 키워드를 더 추가하려면?
**A**: 새 엑셀 파일을 `/home/tlswk/careon/data/row/`에 넣고 `migrate_keyword_data.py` 스크립트를 실행하면 자동으로 분류됩니다.
