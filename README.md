# 케어온 CCTV 고객 분석 & 블로그 자동화 시스템

> "위약금 해방(Un-carrier)" 전략을 위한 페르소나 기반 블로그 자동화 시스템

## 🎯 프로젝트 개요

케어온의 CCTV 렌탈 사업을 위한 키워드 기반 고객 페르소나 분류 및 블로그 콘텐츠 자동화 시스템입니다.

### 핵심 기능

- 🎯 **세부 페르소나 시스템**: 1,692개 키워드를 6개 세부 페르소나로 자동 분류
- 🔥 **3-2 이탈/고통 타겟**: 위약금 고민 고객 14개 키워드 정밀 식별
- 🚀 **퍼널 매핑**: 페르소나별 고객 여정(템플릿/CTA) DB화
- 🤖 **에이전트 협업**: 카피라이터 + 블로그 에이전트 통합 워크플로우

## 🏗️ 시스템 아키텍처

```
[키워드 수집] → [자동 분류] → [세부 페르소나 6개]
                                      ↓
                          [퍼널 전략 매핑]
                                      ↓
              [카피라이터 Gem] ←→ [블로그 에이전트]
                                      ↓
                              [블로그 발행]

                          [템플릿 경로 유도]
                                      ↓
                                  [전환]
```

## 🔥 핵심 전략: "위약금 해방 (Un-carrier)"

### 전략 배경

- **문제**: 전환율 0.38% (월 2,000명 방문 → 7.6건 전환)
- **원인**: 위약금 공포 (50~60만 원 폭탄)
- **해결**: T-Mobile Un-carrier 전략 벤치마킹

### 케어온 오퍼

```
위약금 최대 50만 원 전액 지원
+ 기존 장비 무상 철거
= 월 렌탈료에서 차감

실제 예시:
- 기존 위약금: 48만 원
- 케어온 월 렌탈: 15,000원
- 지원 방식: 매월 10,000원씩 차감 (48개월)
- 실제 부담: 월 5,000원
```

### 타겟 고객

**3-2 이탈/고통 고객 (14개 키워드)**:
- 캡스고객센터 (20,030회)
- 세콤고객센터 (4,140회)
- KT텔레캅고객센터 (2,650회)
- ...

**고객 심리**:
- 기존 업체 불만 (AS, 고객센터)
- 위약금 때문에 갇혀있음
- 갈아타고 싶지만 방법 모름

## 🚀 사용 방법

### 1. 통합 블로그 워크플로우 실행

```bash
python3 scripts/integrated_blog_workflow.py
```

### 2. 실행 흐름

1. 페르소나 선택 (3-2 위약금 해방 타겟)
2. 키워드 선택 (TOP 10 자동 추천)
3. 프롬프트 2개 자동 생성:
   - **STEP 1**: 카피라이터용 (제목 15개 생성)
   - **STEP 2**: 블로그 본문 작성용

### 3. 에이전트 협업

```
카피라이터 Gem (간다 마사노리 카피 단어장)
    → 15개 킬러 제목 생성
블로그 에이전트 (Claude/ChatGPT)
    → 본문 작성 + 퍼널 전략 반영
```
